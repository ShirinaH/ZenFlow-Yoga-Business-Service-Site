# ZenFlow Yoga Studio — SEG3125 Assignment 2

High-fidelity interactive prototype for a local yoga studio service site.

## Tech Stack

- React 18 + Vite
- React Bootstrap + Bootstrap Icons
- React Router

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home with featured classes |
| `/classes` | Filterable class catalogue |
| `/schedule` | Weekly timetable + drop-in policy |
| `/reserve` | 3-step class reservation wizard |
| `/contact` | Contact form with validation |
